# Al-Ghazel

**Work in Progress** - Building an ecommerce bookstore platform.

## About

This project is currently under active development. I am working on transforming this codebase into a full-featured ecommerce platform for a bookstore.

## Tech Stack

This project uses modern technologies provided by the Makerkit template:

- **Next.js 15** with App Router
- **Supabase** for authentication and database
- **Turborepo** for monorepo management
- **TypeScript** for type safety
- **TailwindCSS v4** for styling
- **Shadcn UI** for beautiful components
